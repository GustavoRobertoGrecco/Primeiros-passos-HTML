O que são Clients

Os Clients são os computadores que solicitam as informações aos servidores. Eles apresentam as informações aos usuários através de interfaces gráficas que são criadas por meio da linguagem de programação JavaScript.

O desenvolvimento dessa interface é conhecido como Front-End, ou seja, é a construção do que vai na "frente" do navegador.

O que são Servers

O Server, ou Servidor, é o responsável por armazenar os dados ou arquivos para que os clientes possam requisitar. Esses servidores geralmente são máquinas muito potentes e são destinadas somente para essa função de armazenar.

Os servidores possuem distinções de softwares entre servidores de sistemas operacionais e de servidores Web:

Sistema Operacional:

  • Windows Server
  • Ubuntu
  • CentOS
  • Debian
  • Fedora
  • Oracle Linux

Servidor Web:

  • Apache Http Server Project
  • NGinx

Como é a estrutura básica de um HTML

O HTML possui uma estrutura de marcação composta por três partes principais:

  • < html>< /html>
  • < head>< /head>
  • < body>< /body>

Cada item acima leva o nome tag e elas devem ser "abertas e fechadas" (salvo algumas exceções).

A tag < html> é usada para iniciar e encerrar o documento, ela é considerada a "mãe" das outras tags, sendo hierarquicamente superior.

A tag < head> é onde serão declarados metadados, ou seja, as informações que não serão apresentadas aos usuários da página. Porém, por ser um arquivo HTML, o uso da tag < head> torna obrigatório o uso da tag < title>< /title> em seu interior, onde será inserido o título da página, geralmente apresentado na aba do navegador.

Já a tag < body> é onde será inserido todo o código do que será apresentado ao usuário, ou seja, o "corpo" da página.

Essa é a estrutura básica do arquivo HTML, a partir disso é desenvolvido toda página de um site, contendo inúmeras tags, com as mais variadas funções.

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas levam esse nome pois seguem uma ordem, geralmente numérica, por exemplo:

  1. Esse é o primeiro item
  2. O segundo
  3. Aqui está o terceiro
  4. E assim por diante

Já as listas não ordenadas, por sua vez, não possuem numerações, elas são organizadas através de marcadores:

  • O primeiro item
  • Possui a mesma marcação
  • Que os demais
  • Como tópicos de um mesmo assunto
